## OpenCCK Admin

Nuxt admin panel for OpenCCK CMS

## Build Setup

```bash
# install dependencies
$ npm install

# copy and set environments
$ cp .env.example .env
```

### Development

```bash
# serve with hot reload
$ npm run dev
```

### Production

```bash
# build for production
$ npm run build

# launch with npm
$ npm run start

# launch with pm2
$ pm2 start ecosystem.config.js
```
